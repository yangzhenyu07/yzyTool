# 通用链控工具项目

## 1. 项目介绍
智能合约-链上交互:<br/>
1、多链多节点客户端管理:
基于多租户的概念设计动态链路/节点选择，主要是基于工厂和私有线程实现,<br/>
多节点路由 +  (查询类)加权轮询动态负载均衡 + (交易类)固定链固定节点模式 + RPC健康检查;<br/>
数据流向: 请求头信息 -> 过滤器逻辑 -> 私有线程入参 -> 业务逻辑 -> 客户端相关逻辑 -> 私有线程出参 -> 调用逻辑 -> 过滤器 -> 私有线程清理 <br/>
2、链上交互(交易类)-通用转账;<br/>
3、链上交互(查询类)-账户额度查询<br/>
其他:<br/>
通用excle导出模块、集成https自签证书、基于db集成quartz分布式定时器管理模块，前端代码扫描模块，<br/>
底座(集成swagger、链路日志traceId集成到交互层、自定义异常类架构、支持多端口、令牌控制(校验令牌/刷新领导),<br/>
单元测试集成,flyway数据库集成框架、多数据源、配置文件加密、全局校验),<br/>
自定义手写代码生成,二层缓存设计(本地缓冲+redis缓冲+广播所有本地缓冲失效)。


版本: jdk: 1.8 、SpringBoot 2.3.4
